How to get guilds_api.json

There are several ways to export your Discord server list:

Option 1: Discord API (Recommended)

  1. Open Discord in your browser and log in
  2. Open Developer Tools (F12 or Cmd+Opt+I)
  3. Go to the Network tab
  4. Filter by "guilds" or look for requests to the Discord API
  5. Find a request to /api/v9/users/@me/guilds
  6. Copy the response JSON and save as guilds_api.json

Option 2: Browser Console

  1. Open Discord in your browser
  2. Open Developer Tools Console (F12, then Console tab)
  3. Run the following code:
fetch('/api/v9/users/@me/guilds')
  .then(r => r.json())
  .then(d => {
    const a = document.createElement('a');
    a.href = URL.createObjectURL(new Blob([JSON.stringify(d, null, 2)]));
    a.download = 'guilds_api.json';
    a.click();
  });

Option 3: Legacy Export

If you have a previous export from another Discord tool, it may be compatible if it contains server data with id, name, and features fields.
